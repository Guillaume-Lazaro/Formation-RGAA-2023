Exemples de couples de couleurs avec leur contraste Les couleurs d'écriture et de fond sont chacune présentées sur deux colonnes la première pour indiquer le nom de la couleur et l'autre pour indiquer son code hexadécimal. Enfin la cinquième colonne indique le contraste entre ces deux couleurs.
couleur d'écriture couleur de fond contraste
nom code nom code
darkslategrey #2f4f4f white #ffffff 8,9:1
dodgerblue #1e90ff black #000000 6,5:1
darkturquoise #00ced1 white #ffffff 2:1
